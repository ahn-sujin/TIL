### 2. geolocation API
- ```geolocation API ```는 사용자의 현재 위치 정보를 가져올 때 사용하는 자바스크립트 API이다.
-  ```geolocation API```는 ```navigator.geolocation``` 객체를 통해 사용할 수 있다.
