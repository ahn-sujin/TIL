## 반복문
- 같은 코드를 여러번 반복해야할 경우 사용하는 것이 반복문이다.

<br>

> while문
```javascript
while (???) {
  document.write('1');
  document.write('2');
}
document.write('3');
```
- 이때 ```(???)```에는 불리언 데이터 타입이 들어간다.
- **true**라면 중괄호 안의 코드가 실행된다.
- 코드를 다 실행하고 나면 다시 처음으로 돌아가 ```(???)```에 들어간 불리언 값을 살핀다.
- 불리언 값이 **false**가 될때까지 반복한다.
- 반복문과 조건문은 **순서대로 실행되는 프로그램의 흐름을 제어하는 제어문**이다.

<br>

```javascript
let i = 0;
while(i < 3){
  i = i + 2;
}
document.write(i);

//결과
4 

```
